# Aula 1 - Entendendo o DOM

## O que é o DOM?

### Document Object Model
- O DOM é um padrão de como acessar e modificar os elemento HTML de uma página.


## O que é o BOM?

### Browser Object Model
- O BOM é tudo que está dentro do objeto Window.
